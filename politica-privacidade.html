Política de Privacidade

Compromisso com a proteção dos seus dados pessoais.

A Multiagri Consultoria e Corretora de Seguros LTDA, inscrita no CNPJ sob o nº 06.976.612/0005-24, está comprometida com a proteção dos dados pessoais de seus clientes, colaboradores e parceiros.

1. Coleta de Dados

Coletamos apenas os dados necessários para a prestação dos serviços contratados, incluindo:

  • Nome completo, CPF, data de nascimento e endereço
  • Dados de contato (telefone, e-mail)
  • Informações de saúde para cotação e contratação de planos
  • Dados financeiros para pagamentos

2. Uso dos Dados

Os dados coletados são utilizados exclusivamente para:

  • Cotação e contratação de planos de saúde e previdência
  • Comunicação sobre ofertas e atualizações
  • Envio de documentos e informações contratuais
  • Cumprimento de obrigações legais e regulatórias

3. Compartilhamento de Dados

Seus dados são compartilhados apenas com:

  • Operadoras de saúde e previdência parceiras
  • Órgãos reguladores, quando exigido por lei
  • Prestadores de serviços essenciais (como envio de correspondência)

Importante: Nunca vendemos ou alugamos dados pessoais de nossos clientes.

4. Segurança da Informação

Adotamos medidas técnicas e organizacionais para proteger os dados pessoais contra acesso não autorizado, alteração, divulgação ou destruição.

5. Direitos do Titular

O titular dos dados tem direito a:

  • Acessar seus dados a qualquer momento
  • Solicitar correção de dados incorretos
  • Solicitar exclusão de dados desnecessários
  • Revogar consentimento para uso de dados

6. Contato DPO

Para questões relacionadas à privacidade, entre em contato pelo e-mail: privacidade@multiagri.com.br
